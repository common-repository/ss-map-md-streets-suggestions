Automatically generates street suggestions based on the input values using map.md public API for the Woocommerce checkout page.

== Description ==
Streets Suggestions via map.md for WooCommerce is a plugin that connects map.md with Woocommerce by showing streets suggestions during the checkout process. The streets suggestions are generated based on the input characters from the \"Billing Street address\" and \"Shipping Street address\" fiels.
Streets Suggestions via map.md for WooCommerce is SEO friendly, it loads the .js and .css resources only in the checkout page avoiding to load unnecessary resources in other pages.

The plugin comes already translated into 3 languages which are:
* English 
* Romanian
* Russian

== Installation ==
1. Upload `ss-woo-map-md-suggestions` to the `/wp-content/plugins/` directory
2. Activate the plugin through the \'Plugins\' menu in WordPress
3. Visit the setting page and follow the instructions

== Frequently Asked Questions ==
= Where can I found the map.md API Key? =
The API key should be generated using this page https://map.md/ro/api/
